#  Plants vs Zombies Clone

A browser-based recreation of the classic **Plants vs. Zombies** game developed using **HTML, CSS, and JavaScript**. The project focuses on recreating the core gameplay mechanics with smooth sprite sheet animations and an interactive user experience.

---

##  Features

*  Classic Plants vs. Zombies inspired gameplay
*  Animated zombie movement using sprite sheets
*  Interactive plant placement
*  Collision detection and attack mechanics
*  Smooth CSS animations and transitions
*  Responsive game interface

---

##  Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Sprite Sheet Animations
